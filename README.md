# WanderList

## 项目介绍

这是一个基于React、Redux、React Router、Antd组件库和Electron的待办清单项目，提供桌面端及浏览器端入口。主体功能包含待办事项的添加、删除、编辑、标记完成等操作，同时支持任务分类、搜索、排序等功能。同时系统支持用户自定义配色方案，用户可以根据个人喜好自定义系统的外观。

## 技术栈

1. 前端
  1. React
  2. Redux
  3. React Router
  4. Antd组件库
  5. Electron

2. 后端
  1. Node.js
  2. Express
  3. Mysql数据库


## 功能清单

1. 用户登录、注册功能（手机号+密码，支持验证码登录）
2. 事项类别管理（预置一些类别）
  1. 添加事项类别
  2. 删除事项类别
  3. 编辑事项类别
3. 待办事项管理
  1. 添加待办事项
  2. 删除待办事项
  3. 编辑待办事项
  4. 标记待办事项为完成
4. 数据统计显示
  1. 显示待办事项总数、已完成事项总数、未完成事项总数、事项类别总数
5. 用户自定义配色方案
